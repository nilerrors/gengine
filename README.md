# Gengine

## Description
A simple graphics engine in C++, exporting to bitmap.

## Features
- [x] Intro
- [x] 2D L-System
- [x] 3D Wireframes and 3D L-System
- [x] Z-buffering
- [x] Z-buffering with triangles
- [x] Fractals, (excluding buckyball and mengersponge)
- [x] Ambient Light
- [x] Diffuse Light
- [x] Specular Light
